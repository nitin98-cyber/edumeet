### Method 1: Theme Settings Page
1. Login as admin
2. Click "Themes" in navigation
3. Click any theme preview
4. Done! Theme applies instantly

### Method 2: Direct URL
- Visit: `http://localhost:3000/admin/themes.html`

### Method 3: Batch File
- Double-click `OPEN_THEMES.bat`
